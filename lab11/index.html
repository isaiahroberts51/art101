Lab 11 - Libraries & jQuery

Challenge

The challenge of this lab was to begin to understand how to use the jQuery JavaScript library. We already knew how to perform all of the tasks in this lab with conventional Javascript. The challenge was transferring this knowledge over to jQuery.

Problems

Most of the problems came when we had to implement buttons since we hadn't done so in this fashion before. Creating the button element with jQuery and then creating an event listener for that button was especially difficult.
